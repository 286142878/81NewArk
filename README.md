# 二 ：硬件需求

N卡用户参考表：

|  | 最小配置 | 推荐配置 |
| --- | --- | --- |
| CPU | Intel Core i5-8300H 或同等级别 | Intel Core i7-10875H 或同等级别 |
| 内存 | 8GB | 16GB |
| 显卡 | Nvidia GeForce GTX 1050 或同等级别 | Nvidia GeForce GTX 1660 Ti 或同等级别 |
| 存储 | 256GB SSD | 512GB以上 SSD  |
